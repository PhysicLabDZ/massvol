# محاكاة كثافة الجسم الصلب - PhysicLabDZ 🧪

أداة تعليمية تفاعلية مصممة لتبسيط مفهوم الكثافة والطفو للأجسام الصلبة في السوائل المختلفة، موجهة لطلاب المدارس وهواة الفيزياء.

## 🌟 نظرة عامة
هذا المشروع عبارة عن تطبيق ويب (PWA) يعتمد على **HTML5 Canvas**، يتيح للمستخدم إجراء تجارب افتراضية لقياس كثافة المواد المختلفة ومراقبة سلوكها (الطفو أو الغوص) بناءً على القوانين الفيزيائية الحقيقية.

## ✨ المميزات
- **تجربة تفاعلية:** سحب وإفلات الأجسام داخل حوض السوائل.
- **تنوع المواد:** تجربة مواد مختلفة (جليد، خشب، حديد، فلين، بلاستيك، نحاس).
- **تغيير البيئة:** إمكانية تغيير السائل (ماء، زيت، كحول، زئبق) لملاحظة فرق النتائج.
- **حسابات فورية:** عرض الكتلة، الحجم، والكثافة بشكل حي أثناء التجربة.
- **تصميم متجاوب:** يعمل بكفاءة على الحواسيب والأجهزة اللوحية والهواتف الذكية.
- **دعم PWA:** يمكن تثبيت التطبيق على الهاتف ليعمل كتطبيق مستقل.

## 🛠 التكنولوجيات المستخدمة
- **HTML5:** لبناء الهيكل الأساسي.
- **CSS3:** للتنسيق واستخدام الخطوط العربية (Tajawal).
- **JavaScript:** للبرمجة المنطقية، الحسابات الفيزيائية، ورسم العناصر على `Canvas`.
- **Canvas API:** لعمل المحاكاة البصرية والتحريك.

## 🚀 كيفية التشغيل
لا يحتاج هذا المشروع إلى خادم (Server) خاص، يمكنك تشغيله ببساطة عبر:
1. قم بتحميل الملفات أو عمل `git clone` للمستودع.
2. افتح ملف `index.html` في أي متصفح ويب حديث.

## 📂 هيكل الملفات
- `index.html`: الملف الأساسي الذي يحتوي على الكود البرمجي والتنسيق.
- `manifest.json`: ملف التعريف لتحويل المشروع إلى تطبيق ويب تقدمي.
- `assets/`: (اختياري) إذا كنت ستضيف صوراً أو أيقونات لاحقاً.
